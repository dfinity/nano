# quill scanner-qr-code

Print QR Scanner dapp QR code: scan to start dapp to submit QR results.

## Basic usage

The basic syntax for running `quill scanner-qr-code` commands is:

``` bash
quill scanner-qr-code
```

## Flags

| Flag                 | Description                                     |
|----------------------|-------------------------------------------------|
| `-h`, `--help`       | Displays usage information.                     |
